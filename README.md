# fumes
This projects aims to create a sleek and minimalist fume extractor. By that i mean a fume extractor that is one solid shape without an extra base or any elements sticking out. It includes an activated carbon filter on the front and a HEPA filter on the back to ensure clean air in my room :)
# BOM
- Noctua NF-F12 industrialPPC-3000 PWM, 120 mm (~30 USD)
- HEPA filter sheets (~10 USD)
- activated carbon filter sheets (~10 USD)
- 3D printed enclosure (/)
- USB Type C Power Delivery Breakout Board (~ 6 USD)
